Installation
============

Requirement
-----------

* Python >= 2.7
* Gtk >= 3.0
* PyGObject module


Setup
-----

Download from https://github.com/formateli/gpropertygrid

Once unziped, on project directory:

Install::

    python setup.py install --record files.txt

Uninstall (Ubuntu)::

    sudo bash -c "cat files.txt | xargs rm -rf"
